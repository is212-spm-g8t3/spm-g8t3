Courses Related

courses

Description
Returns a list of courses in JSON format.

HTTP request
GET http://../courses

get active courses by skill_id

Description
Returns a list of active courses that fulfil a specific skill in JSON format.

HTTP request
GET http://../getCoursesBySkill/<skillID>
where skillID is the ID of a specific skill.

get skills satisfied by course

Description
Returns a list of skills that is fulfilled a specific course in JSON format.

HTTP request
GET http://../getCourseSkills/<course>
where course is the course name of a specific course.

get all courses by skill_id

Description
Returns a list of courses that fulfil a specific skill in JSON format.

HTTP request
GET http://../getCourseSkills/<skill_id>
where skill_id is the skill id of a specific skill.

Roles Related

get all job roles

Description
Returns a list of all job roles in JSON format.

HTTP request
GET http://../roles

get all job roles with their associated skills

Description
Returns a list of all job roles and their associated skills in JSON format.

HTTP request
GET http://../getRolesWithSkills

create role

Description
Creates a role in the database

HTTP request
POST http://../createRole
where POST data parameter name is the name of the role to be created, description is the description of the role and department is the department the role belongs to.

Skills Related

get all skills

Description
Returns a list of all skills in JSON format.

HTTP request
GET http://../skills

get skills by job role ID

Description
Returns a list of skills base on the provided job role ID in JSON format.

HTTP request
GET http://../skills-by-role/<roleId>
where roleId is the role ID of the role to be queried.

add new skill

Description
Creates a skill in the database

HTTP request
POST http://../addNewSkill
where POST data parameter name is the name of the skill to be created, description is the description of the skill , type is the type the skill has and status is the status of the skill.
