LJPS API v1.0 - Documentation

itineraries

Description
Returns the itineraries of a user in JSON format.

HTTP request
GET http://../database_connectivity/itineraries.php?username=alice
where username is the username parameter, and alice is the user value.
In the case of local API installation, the exact URL may vary - please check your local setup.

add_itinerary

Description
Adds an itinerary to the database.

HTTP request
GET http://../database_connectivity/add_itinerary.php
Work in Progress
In the case of local API installation, the exact URL may vary - please check your local setup.

remove_itinerary

Description
Remove an itinerary from the database based on specified itinerary id.

HTTP request
GET http://../database_connectivity/remove_itinerary.php?itinerary_id=2
where itinerary_id is the itinerary_id parameter, and 2 is the itinerary_id value.
In the case of local API installation, the exact URL may vary - please check your local setup.

events

Description
Returns events of an itinerary in JSON format.

HTTP request
GET http://../database_connectivity/events.php?username=bob&itinerary_name=test
where username is the username parameter, and bob is the username value and where itinerary_name is the itinerary_name parameter, and test is the itinerary_name value.
In the case of local API installation, the exact URL may vary - please check your local setup.

remove_event

Description
Remove an event from a specified itinerary id based on the event id.

HTTP request
GET http://../database_connectivity/remove_event.php?itinerary_id=4&id=5
where itinerary_id is the itinerary_id parameter, and 4 is the itinerary_id value and
where id is the event id parameter, and 4 is the event id value
In the case of local API installation, the exact URL may vary - please check your local setup.
